2023 스마트농업 AI 경진대회 해커톤을 위한 Repo입니다.

# 해커톤(예선) 문제
- 정식일 이후 16주 동안 단위면적당 최소 에너지 비용(누적 난방용 도시가스 사용량)으로 최대 수확량을 예측 할 수 있는 환경/생육 모델을 구하세요.
- 모델의 성능과 정확도 평가는 학습 모델의 RMSE 와 R2 Score 로 평가 합니다.
